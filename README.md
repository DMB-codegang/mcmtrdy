## 登录服务器

### 教程
自2025年10月26日起，服务器采用密码+QQ账号绑定登录。如果您不清楚如何登录，请参照以下内容操作：

> 先决条件:
> 您已经加入了服务器的QQ群

1. 使用启动器启动MC并进入服务器。
2. 如果之前您没有绑定过QQ号，您在进入服务器的时候将会被踢出。请留意观察信息，**找到验证码并记住它**
3. 打开QQ，找到服务器的QQ群，并在**QQ群**中发送`mcl bind <验证码>`。请将`<验证码>`替换为您在第2步中找到的验证码。
4. 检查机器人发送的账号名是否与您的游戏账号名一致。如一致，请发送“确认”以确认绑定。
5. 现在你已经成功绑定了您的QQ号。重新进入服务器。
6. 如果您是初次游玩，服务器会要求您设置一个密码保障账户的安全。请使用指令`/register <密码> <确认密码>`设置密码。请将`<密码>`和`<确认密码>`替换为您需要设置的密码，并确保两次输入的密码一致。该密码将会在您后续登录时使用。
7. 下次服务器要求登录时，请使用指令`/login <密码>`或`/l <密码>`登录。请将`<密码>`替换为您在第5步中设置的密码。

## 常见问题

### 验证QQ时没有出现验证码而是提示`Internal Exception:Java.net.SocketException:Connection reset`

请重新进入服务器以尝试重新获取，验证码从你第一次进入服务器内开始算起3分钟内不会变动。如果该问题一直持续出现，请检查网络连接。

### 验证QQ时机器人提示`验证码无效`

请检查您输入的验证码是否正确。如果依然不行，请进入服务器尝试重新获取验证码。

### 忘记密码了怎么办

请联系管理员重置密码，使用qq重置密码的功能正在开发

如果在使用过程中遇到其他问题，请联系服务器管理员。
